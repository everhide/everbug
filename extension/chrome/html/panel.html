      Welcome to everbug!



      Installation

      As usual:

      Run "pip install everbug"

      Add "everbug" to your INSTALLED_APPS.

      Finally, append "everbug.middleware.Tracer" to MIDDLEWARE or MIDDLEWARE_CLASSES in settings.py.
